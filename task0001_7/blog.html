CSS

css中需要注意的问题

1.适当地使用position：relative和position：absolute，需要绝对定位的元素，其父元素不要忘了设置相对定位，子元素才能随着屏幕的缩放而移动。

2.当给内容改变padding值时，整个div的大小也会发生变化，需要在改变padding时适当调整div宽高。

3.一些标签如p、h等默认有上下边距，在标签嵌套标签时，当布局发生问题或微小偏移，考虑是否是margin产生的。

4.在嵌套的标签中，里层的某个类名即使是独一无二的，也尽量使用关联选择器的写法，以保证优先级最高，不会受到外层样式的影响。

5.div高度自适应文字内容高度：最内层div不设置高度，最外层div都设置overflow:hidden（由于内层元素浮动导致外层不能自动撑开).

6.盒子根据内容自动向上撑开的方法：不设置盒子高度，并用绝对定位固定bottom值。

CSS

css中的几个小技巧

1.嵌套的div中，子集设置了margin-top，若父级未设置，父级也会默认为该margin-top，可以给父级设置border-top:1px或者padding-top:1px来阻隔父级受到子集的影响。

2.几块浮动的div高度一致且等于其中内容撑开最高的一块div的高度的方法：对div设置padding-bottom：1000px; margin-bottom: -1000px（1000非固定值，可设置更大）。

3.若干浮动元素影响了布局时，适当用magin-top，margin-left来调整到合适位置。

CSS

css Get新技能

1.点击输入框时去掉默认的外框效果的方法：设置outline:0；焦点停在输入框时，默认value值消失的方法：将value替换为placeholder。eg:input type="text" placeholder="Keywords"。

2.背景渐变以及半透明的方法。eg:background: -webkit-gradient(linear, 0% 0%, 0% 100%, from(rgba(256,256,256,0.3)), to(rgba(256,256,256,0.1)));

3.div部分设置背景色（如排行榜的效果）。eg:background:-webkit-gradient(linear, 0% 0%, 100% 0%, from(#522a5c), to(#fff), color-stop(0.6,#522a5c), color-stop(0.6,#fff));

4.圆角矩形：border-radius: 30px；圆形边框：border-radius: 50%。

5.背景图自适应屏幕宽高：background-size: contain；背景图位置：background-position: center top;（第一个值左右，第二个值上下）。
